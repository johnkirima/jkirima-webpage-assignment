# John Kirima — Personal Landing Page

A minimal, brutalist personal landing page built from scratch with vanilla HTML, CSS, and JavaScript.

## About

This site serves as a professional portfolio and digital business card for John Kirima — a Business Analytics & Information Systems student at the University of Iowa (Class of 2026). It showcases skills, projects, and experience in data science and analytics.

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — External stylesheet, no frameworks
- **JavaScript** — Minimal, vanilla JS for smooth scrolling
- **Google Fonts** — Space Mono (monospace)

## Design

- Brutalist / terminal aesthetic
- Dark theme (`#050505` background)
- Monospace typography throughout
- Fully responsive (mobile-first)
- WCAG 2.2 Level AA accessible
